## Run Locally

Go to the project directory 

```bash
  cd Zubee/frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Go to the project directory 
```bash
  cd Zubee/backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
